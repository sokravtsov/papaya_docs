# Installation

***

### Package Manager <a href="#package-manager" id="package-manager"></a>

Install the required packages.

{% tabs %}
{% tab title="npm" %}
```bash
npm install @papaya-finance/sdk
```
{% endtab %}

{% tab title="yarn" %}
```bash
yarn add @papaya-finance/sdk
```
{% endtab %}

{% tab title="pnpm" %}
```bash
pnpm add @papaya-finance/sdk
```
{% endtab %}
{% endtabs %}
